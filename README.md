## React Fundamentals Learning Journey

This repository is a collection of projects and exercises focused on mastering **React** and its core principles. It serves as a personal laboratory for exploring technical implementations, hooks, and state management.

---

### ### Repository Overview

The following modules cover essential React concepts, moving from basic UI components to complex state logic and hooks:

*   **Core Concepts & UI**
    *   `card-components`: Building reusable UI structures.
    *   `props`: Passing data between components.
    *   `conditional-rendering`: Handling logic-based UI changes.
    *   `render-lists`: Efficiently mapping through data arrays.
    *   `css-styling`: Implementing various styling methodologies.
*   **Event Handling & State**
    *   `click-events` & `onchange-event-handler`: Managing user interactions.
    *   `use-state-hook`: Managing local component state.
    *   `updater-functions`: Safely updating state based on previous values.
    *   `update-objects-in-state` & `update-arrays-in-state`: Mastering immutable state updates.
    *   `update-array-of-objects-in-state`: Handling complex data structures in React.
*   **Advanced Hooks**
    *   `use-effect-hook`: Managing side effects and lifecycle events.
    *   `use-context-hook`: Handling global state without prop-drilling.
    *   `use-ref-hook`: Accessing DOM elements and persisting values without re-renders.
*   **Mini Applications**
    *   `color-picker-app`
    *   `digital-clock-app`
    *   `stop-watch-app`
    *   `todo-list-app`

---

### ### Philosophy

This repository reflects a "perfectionist" approach to learning—focusing on understanding how these tools work "under the hood" rather than just surface-level implementation. Each folder represents a step toward a deep conceptual understanding of modern full-stack web technologies.
